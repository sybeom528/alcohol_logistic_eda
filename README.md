# alcohol_logistic_eda

본 프로젝트는 FAF5 물류 데이터를 활용하여  
주류(맥주/브랜디) 사업 전략을 다음과 같은 순서로 제시한다.

1. 주류 공장 설립

2. 주요 운송 주 선정

3. 브랜디 수출 전략

### 1. 주류 공장 설립 분석

- 원료 수급이 유리하면서, 주로 외부 주에서 주류의 공급을 받는 주를 선정

1. tons_orig - tons_dest로 net_outflow 파생 변수를 생성해 공급이 많은 주를 선정

![net_outflow 상위 10개 주](images/net%20outflow%20of%20top10.png)

2. tons_orig로 원료 생산지를 파악

![미국 주요 곡물 생산지](images/미국%20주요%20곡물%20생산지.png)

![미국 기타 농산물 생산지](images/미국%20기타%20농산물%20생산지.png)

3. 지도 시각화

![미국 주별 주류 수급 현황](images/미국%20주별%20주류%20수급%20현황.png)

결론 : 주류를 외부 공급에 의존하면서도, 원료 수급이 유리한 일리노이 주를 선정해 운송비 절감 및 일리노이 공급 계약 난이도 하락

### 2. 주요 운송 주 선정

- 외부 공급에 의존하는 대도시 주를 선정(net outflow < -500, 인구 1,000,000만 이상)하고, 단가(value/ton)을 활용해 맥주, 브랜디 운송 전략 도출

![미국 주별 주류 단가](images/미국%20주별%20주류%20단가.png)

1. 위의 기준에 부합하는 주를 지도에 표시

2. 24년 맥주 주류 단가는 0.78 ~ 2.29달러로 형성 -> 2.29 이상의 주에 추가적으로 브랜디 운송

결론 : 워싱턴을 제외한 대부분의 주가 동부에 형성되어 동부에 주류 판매 집중 및 뉴욕, 플로리다, 매사추제츠에 브랜디 판매

### 3. 브랜디 수출 전략

1. 미국 수입/수출 시장 분석

| 주류 수입 트렌드(톤) | 주류 수출 트렌드(톤) |
| :---: | :---: |
| ![그래프1](images/주류%20수입%20트렌드.png) | ![그래프2](images/주류%20수출%20트렌드.png) |

수입에 비해 수출 시장은 경쟁력이 과도하지 않을 것으로 판단해 브랜디 수출 결정

2. 주류 주요 수출국 트렌드 분석

| 주요 수출국 달러 트렌드 | 주요 수출국 톤 트렌드 |
| :---: | :---: |
| ![그래프1](images/미국%20주요%20수출국%20달러%20트렌드.png) | ![그래프2](images/미국%20주요%20수출국%20톤%20트렌드.png)

중동은 달러가 증가하지만, 톤은 유지되고 있음 -> 주류의 단가가 증가하고 있음

![중동 단가](images/미국%20주요%20수출국%20단가%20트렌드.png)

가격의 증가 추이를 보았을 때, 고급주의 취급을 늘리는 것으로 예상되므로, 주요 수출을 중동으로 선정

3. 운송 허브 거점 선정

![중동아시아 주요 수출 거점](images/중동아시아%20주요%20수출%20거점.png)

일리노이 주와 가장 가까우면서도, 현재 중동에 주류를 수출하는 거점인 테네시 주를 수출 운송 허브로 선정

![미국 수출 거점 운송수단](images/미국%20수출%20거점%20운송수단.png)

추가적으로, VIP 고객 확보를 위한 항공 운송을 위해 플로리다를 항공 운송 허브로 선정
